# Using 42Coin  
### Transfer Tokens  
1. In MetaMask, click `Send` and enter the recipient's address.  
2. Amount: `1` = 1 full 42C (auto-scaled to `1 * 10^18` units).  

### Add 42C to MetaMask:  
- **Contract Address**: `0x2Bf4f83B77F147811a6dB52A4210257F46B0c79C`  
- **Symbol**: `42C`  
- **Decimals**: `18`
